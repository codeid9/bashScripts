# Bash Scripts


## basic of bash

- read: used to take input
- -p is used to print msg while input
- -t is used to input something within given time 
- -s for silent input line password
- -a is used to take input in array






### dirgen.sh

Using this script you can create a Directory.
After creating first directory it will ask you to create sub-directories (y/N).

### sys_health_check.sh

this script will show you ip address,system memory and system uptime.

### greeting

this script will greet you and print date on terminal.
